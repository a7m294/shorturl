#### nginx rewrite
```
nginx/1.10.1
```
```nginxconf
root   /home/[your user]/[project name]/shorturl/public_html;

location / {
    rewrite ^/$ /index.php last;
    rewrite ^(.*)$ /shorturl.php?go=$1 last;
    rewrite ^(.*)\.php$ /shorturl.php?go=$1 last;
}
```
